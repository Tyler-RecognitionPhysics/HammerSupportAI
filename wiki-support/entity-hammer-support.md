# Hammer customer support

Hammer provides AI-powered lead response and follow-up for car dealerships. Existing customers use Hammer Drive, Facebook AIA, MarketPoster, and Hammer Connect.

## Contact Hammer support

- **Email:** support@hammertime.com
- **Phone:** (512) 883-1336
- **Hours:** Monday–Friday, 9:00 a.m. – 5:00 p.m. Central Time

## Common support areas

### Login and account access

- Password reset via Welcome email / account activation flow
- OTP login via SMS short code 426637
- Account managers can help with access issues — escalate if self-service fails

### Hammer dashboard

- Lead inbox, conversation threads, reporting metrics
- Settings for dealership profile, users, and integrations
- SMS opt-out: reply STOP or remove number from dashboard

### Integrations (CRM / DMS)

- DealerTrack, Tekion, VinSolutions, CDK, and others — setup varies by rooftop
- Integration issues often need account manager or support team — gather dealership name and CRM name before escalating

### Facebook AIA

- Meta ad inventory sync, sponsored listings, lead routing into Hammer
- Billing: flat monthly fee plus Meta daily minimum spend (see account contract)

### MarketPoster

- Chrome extension for posting inventory to Facebook Marketplace
- Listing sync and posting frequency settings

### Hammer Connect

- Marketplace messaging routed into Hammer; SMS thread continuity

### Billing and subscription

- Month-to-month billing on file after activation
- Renewal and card updates through account settings or support team

## Escalation

If troubleshooting steps in the knowledge base do not resolve the issue, offer human support contact above and note the issue category for follow-up.
